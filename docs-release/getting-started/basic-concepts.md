# Basic Concepts

This guide provides an overview of basic concepts to ensure a successful onboarding experience.

## Prerequisites

- Read the Quickstart guide.
- Understand standard project directory structures.

## Steps

1. **Understand Sprints**: A sprint is a bounded unit of work.
2. **Review Orchestration**: The orchestrator handles task assignment and prioritization.
3. **Use the Dashboard**: Manage everything from the local dashboard on port 4444.

## Expected Result

You will have a clear understanding of the core concepts needed to operate the system effectively.
